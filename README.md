# 🍕 Pizza House — Next.js застосунок

Pizza House — це сучасний веб-сайт піцерії, створений на **Next.js 15** та **React 19**.  
Проєкт містить сторінки меню, оформлення замовлення, промо-банери та інші маркетингові блоки.

---

## 🚀 Стек технологій

- **Next.js 15** — фреймворк для React з рендерингом на сервері та маршрутизацією
- **React 19**
- **TypeScript 5**
- **Material UI (MUI)** `@mui/material`, `@mui/icons-material`
- **Emotion** `@emotion/react`, `@emotion/styled` — стилізація MUI
- **styled-components** — додаткові стилі компонентів
- **Tailwind CSS** — утилітарні класи для швидкої верстки
- **Redux + React Redux** — глобальний стан (кошик, користувач тощо)
- **Zustand** — легкий стейт-менеджер для локального/ізольованого стану
- **Formik + Yup** — форми (оформлення замовлення, контакти) та валідація
- **react-slick + slick-carousel** — слайдери (банери, акції, відгуки)
- **Puppeteer** — автоматизація браузера (наприклад, для генерації PDF/скріншотів)

---

## 📂 Основні можливості (приклад)

> За потреби зміни цей список під свої реалізовані фічі.

- Перегляд меню піц з описом та цінами
- Додавання піци в кошик та зміна кількості
- Оформлення замовлення через форму (Formik + Yup)
- Слайдери з акціями/хітами/відгуками (react-slick)
- Адаптивна верстка на базі MUI + Tailwind/styled-components
